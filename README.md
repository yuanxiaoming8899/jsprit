<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">jsprit 是一个基于 Java 的开源工具包，用于解决丰富的</font></font><a href="http://en.wikipedia.org/wiki/Travelling_salesman_problem" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">旅行商问题 (TSP)</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和</font></font><a href="http://neo.lcc.uma.es/vrp/vehicle-routing-problem/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">车辆路径问题 (VRP)</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。它轻量、灵活且易于使用，并且基于一个</font><font style="vertical-align: inherit;">目前用于解决的</font></font><a href="/graphhopper/jsprit/blob/docs/Meta-Heuristic.md"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">通用元启发式算法</font></font></a><font style="vertical-align: inherit;"></font></p>

<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">容量受限的车辆路径规划</font></font></li>

<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">带时间窗的 VRP</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">带回程的 VRP</font></font></li>

<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">异构车队的 VRP</font></font></li>
